# Frac Stacker

Fractional silver collectibles - 1 gram rounds for collectors and gifts.

## Overview

Frac Stacker offers premium 1 gram silver rounds for two audiences:

- **Frac Stacker** - For collectors building their silver stack
- **Silver Grams** - For gifting meaningful silver with custom messaging
